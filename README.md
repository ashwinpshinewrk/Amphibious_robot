# AmphiBot

This repo contains all the software , hardware and diagrams.

Built on ROS2 Jazzy.

## Quick Start

You can build this with these steps, but make sure to install ROS2 jazzy, g++ and other dependencies of ROS2.


```bash
clone the repo. (by git clone)
cd <to the folder>
colcon build
source install/setup.bash
# Todo add master file to run
```

## Repo Structure.

- `/docs` : Contain all project documentation , reports
- `/hardware` : Contain all CAD files, electronic schematics and datasheets
- `/src` : ROS2 source directory with all packages
